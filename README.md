# dnyaneshwariraut.com

Personal portfolio of **Dnyaneshwari Raut** — AI Engineer · ML Engineer · AI Software Engineer.

Designed and built from scratch (no templates): Next.js App Router, Tailwind CSS v4, and Framer Motion, with an animated agent-graph hero rendered on canvas, original hand-drawn SVG illustrations for every project, and a custom "DR" graph monogram.

## Stack

Next.js 16 (App Router) · TypeScript · Tailwind CSS v4 · Framer Motion · next-themes

## Develop

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
```

## Notes

- Dark-mode-first with a light toggle (`next-themes`, class strategy).
- All project visuals are original SVGs in `src/components/illustrations/` — no stock assets.
- Resume: drop the final PDF at `public/assets/resume/resume.pdf`. The Download Resume buttons already point there.
- Photos live in `assets/photos/` and are statically imported through `next/image`.
